\section{What Is Aion?}

Aion is a self-hosted personal AI assistant built on a local LLM (Ollama / qwen2.5:7b),
deployed on Vast.ai GPU cloud instances. It features a web chat interface,
persistent multi-user memory, voice output (ElevenLabs TTS), and an admin panel
for managing cloud deployments — completely private with no third-party data retention.

\begin{callout}
  \textbf{Production URL:} \href{https://drayhub.org}{https://drayhub.org} \quad
  \textbf{LLM:} qwen2.5:7b via Ollama \quad
  \textbf{Platform:} Vast.ai GPU cloud
\end{callout}

% ── Key Features ──────────────────────────────────────────────────────────────
\section{Key Features}

\subsection{Multi-User Authentication}
\begin{itemize}[leftmargin=1.2em, itemsep=2pt]
  \item Login with username and password (bcrypt hashed, stored in SQLite)
  \item 30-day sliding session cookie (\code{aion\_token}, httponly)
  \item Roles: \textbf{admin} and \textbf{user}; Brian's admin account auto-created on first run
  \item Per-user chat history stored in SQLite — last 40 turns sent to LLM as context
  \item Cross-device: same account works from any browser or device
\end{itemize}

\subsection{Shared Persistent Memory}
\begin{itemize}[leftmargin=1.2em, itemsep=2pt]
  \item All users share one fact store (\code{shared\_learned.jsonl})
  \item Facts learned by Brian are visible when any other user logs in
  \item Explicit shortcut: \code{remember: Gary is Brian's friend from Laconia} — instant, no LLM call
  \item TF-IDF retrieval injects the 15 most relevant facts into every system prompt
  \item Persists across server restarts — backed by file on disk
\end{itemize}

\subsection{Auto Fact Extraction}
\begin{itemize}[leftmargin=1.2em, itemsep=2pt]
  \item After every LLM reply, a daemon thread asks the LLM to extract notable facts
  \item Extracted facts are added to shared memory automatically
  \item Zero latency impact — fire-and-forget, user gets their reply immediately
\end{itemize}

\subsection{Memory Browser}
\begin{itemize}[leftmargin=1.2em, itemsep=2pt]
  \item Full Facebook message browser built into the chat UI
  \item Categorised by: Birth \& Pregnancy, Love \& Relationships, Family \& Parenting,
        Health \& Wellbeing, Loss \& Grief, Major Life Events, General
  \item Expandable conversation threads rendered as chat bubbles with timestamps
\end{itemize}

\subsection{Vast.ai Admin Panel (\texttt{/admin})}
\begin{itemize}[leftmargin=1.2em, itemsep=2pt]
  \item \textbf{GPU Market:} search available instances by max \$/hr, min VRAM, GPU name
  \item Shows GPU model, VRAM, system RAM, disk, network speed, reliability \%, and price
  \item One-click \textbf{Deploy} — provisions instance and runs full auto-deploy script:
    installs Ollama, pulls qwen2.5:7b, clones repo from GitHub, installs Python deps,
    starts Aion on port 5000
  \item \textbf{Running Instances:} shows status, SSH info, web URL, uptime, and hourly cost
  \item \textbf{Redeploy Code} — rsync local changes and restart gunicorn via SSH
  \item \textbf{Destroy} — terminates instance immediately
\end{itemize}

% ── Security ──────────────────────────────────────────────────────────────────
\section{Security}

\begin{itemize}[leftmargin=1.2em, itemsep=2pt]
  \item All API keys and passwords live in \code{config\_local.py} (gitignored — never pushed)
  \item \code{data/} directory is gitignored — all private family data stays local only
  \item \code{.gitattributes} enforces LF line endings; \code{core.autocrlf=input} set globally
  \item \code{/admin} route protected server-side by role check before rendering
  \item Session tokens are 32-byte hex with 30-day sliding expiry; httponly cookie
  \item \textbf{Action required:} ElevenLabs key was previously committed to git history —
        rotate it at \href{https://elevenlabs.io}{elevenlabs.io}
\end{itemize}
